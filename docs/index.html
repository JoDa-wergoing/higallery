HiGallery

HiGallery is a WordPress plugin that displays photo albums directly from Strato HiDrive. Images are loaded from external storage and presented in a modern, responsive gallery using PhotoSwipe v5.

No media uploads to WordPress, no duplication, and no unnecessary storage usage.

---

What is HiGallery?

HiGallery connects WordPress to your HiDrive account using secure OAuth authentication. Photo albums are read directly from HiDrive folders and rendered as responsive galleries inside WordPress pages or posts.

This makes HiGallery ideal for websites that want to keep media files outside WordPress while still offering a fast and user-friendly gallery experience.

---

Key features

  • Display photo albums directly from Strato HiDrive
  • No media stored or duplicated in WordPress
  • Modern PhotoSwipe v5 lightbox (mobile & retina optimized)
  • Responsive image loading with smart resizing
  • Secure OAuth authentication
  • Clean integration with WordPress pages and posts
---

Why use external storage for WordPress galleries?

  • Keep WordPress backups small and fast
  • Avoid media library clutter
  • Reuse the same photo albums across multiple websites
  • Better separation between content and storage

HiGallery is specifically designed for users of Strato HiDrive who want a reliable and maintainable gallery solution for WordPress.
